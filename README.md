## Getting Started

Clone this repository:

```bash
git clone https://github.com/devmikemwai/react-loginui.git
cd react-loginui
```

Install dependencies:

```bash
npm install
```

Run the app:

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
